# Content Aggregator Frontend

This is a React frontend for the Content Aggregator application. It allows users to view articles collected from various news sources, manage those sources, and trigger the scraping process.

## Bootstrap Integration

This project uses React Bootstrap for its UI components. Bootstrap provides a comprehensive set of responsive components that make it easy to create a professional-looking interface.

## Setup and Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend/main-app
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Key Features

- View articles from various news sources
- Manage news sources (add, edit, activate/deactivate)
- Trigger content scraping operations
- Responsive design using Bootstrap

## Project Structure

- `src/components/` - React components
- `src/services/` - API service for backend communication
- `src/assets/` - Static assets
- `public/` - Public assets

## Technologies Used

- React 19
- React Router v7
- React Bootstrap
- TypeScript
- Vite
- Axios for API calls
